# Stress Calendar

## Created By - Chipot-lovers



### **Structure**

### Built using Spring Boost framework and supabase + postgresql dependencies for database.

### Uses a Controller, Model, Repository, and Service structure to interact with an HTML + CSS + JavaScript frontend and a Java backend.

### Database makes use of local UUIDs to save information between launch sessions.

### 

### Program allows user to organize events on a calendar with daily, weekly, and monthly views. Each day has a stress level value that allows users to view average stress levels during the day, week, and month.
